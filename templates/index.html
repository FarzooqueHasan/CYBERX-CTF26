Challenge Overview

Welcome to the official CyberX Club Induction CTF. This pipeline tests foundational skills in cybersecurity across four distinct categories:

  • Recon & Log Analysis: Directory enumeration and log parsing.
  • Decryption & Key Derivation: Cryptographic key expansion and symmetric payload decryption.
  • Identity & Access Control: Inspecting network APIs and identifying privilege escalation flaws.
  • Image Steganography Analysis: Bitwise LSB extraction using custom Python scripts.
Notice: Out-of-bounds scanning, directory brute-forcing, and SQL injection are strictly prohibited. All answers are derived from logical analysis.
